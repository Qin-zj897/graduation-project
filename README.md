# graduation-project
毕业设计
20260316-20260322
静态分析部分
1、输入结构识别，新增了一个方法，输出变量的数量、每个变量的输入入口（行号）、类型、对应哪个变量
2、将CFG 分支，谓词，约束唯一标识绑定，把CFG 分支节点、谓词、true/false 两个分支需要满足的约束一起输出，主键是branch_id
3、将变量的候选值进行聚合，具体的格式为
{
  'variable'         : 输入变量名
  'input_type'       : 输入类型
  'input_format'     : 输入格式
  'branches'         : [分支id、候选值列表]
}
4.删除了后向切片相关方法
动态分析部分
1、增加了增量式统计：每次进行测试时输出哪些是新增覆盖、哪些分支还没有覆盖
2、将分支改为与谓词 ID绑定
3、增加了输出中每个变量到分支的变量传递链路
4、增加了for循环结构的分支距离计算，能进入循环体时距离为0，迭代器耗尽时距离为1
静态分析的测试在test_simple，简单地将各个方法的输出打印在控制台；动态分析的测试在test_dynamic，输出到文件夹test_result中
